# Notebooks

This folder contains two examples:

* [`step1_train_on_cutout.ipynb`](./step1_train_on_cutout.ipynb): training a simple binary classifier on a scene cutout;
* [`step2_prediction_on_large_data.ipynb`](./step2_prediction_on_large_data.ipynb): apply the trained model on a larger RGB mosaic.

Beside these, [`data_preparation/data_preparation.ipynb`](./data_preparation/data_preparation.ipynb) illustrates the process of downloading and generating the example datasets.
